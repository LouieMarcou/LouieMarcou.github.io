Mac Version: Untiled Roguelike

A 2D Roguelike game that is a work in progress. This is a side project and not a requirement for any class.

Windows download instructions: Unzip it and open the 2DActionRoguelike.exe file.

Mac download instructions: Unzip the file, open up a terminal and go to the app folder:

cd path_to_your_app/RoguelikeGameBuildMac.app/Contents/MacOS/

Now to make it executable type in:

chmod +x RoguelikeGameBuildMac

And now you should be able to start the app.
